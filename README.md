
# BCA CC Lab – 2nd Year

Cloud Computing Lab Programs and AWS Experiments (4th Semester)

## Cloud Computing Lab Experiments (AWS)

This repository contains AWS Cloud Computing laboratory experiments performed using various AWS services such as EC2, S3, RDS, EBS, and EFS.

## Experiments Included

1. Launching Linux and Windows Virtual Machines using Amazon EC2  
2. Launching a MySQL Database using Amazon RDS  
3. Hosting a Static Website using Amazon S3  
4. Speeding Up Data Transfer using S3 Transfer Acceleration  
5. Cross Region Replication using Amazon S3  
6. Hosting a Static Website using Amazon EC2  
7. Creating a Custom Amazon Machine Image (AMI)  
8. Allocating an Elastic IP to an EC2 Instance  
9. Attaching Amazon EBS and Amazon EFS to EC2  
10. Creating Snapshots of EC2 / EBS Volumes  

## Technologies Used

- Amazon EC2
- Amazon S3
- Amazon RDS
- Amazon EBS
- Amazon EFS
- Linux
- Windows Server
